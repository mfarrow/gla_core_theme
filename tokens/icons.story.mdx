# Icons

When adding any new icons please remember to:

- Trim the SVG canvas size to match the dimensions of the shape inside (the
  _Document Properties_ dialog in [Inkscape](https://inkscape.org) is useful for
  this)
  - If you use Inkscape make sure to save the SVG as an 'Optimized SVG' not an
    'Inkscape SVG'
- Remove any dimensions but leave the `viewBox` to help the SVG scale
- Remove explicit colour references and replace with `currentColor`
- Optimise/minify using SVGO or
  [SVGOMG](https://jakearchibald.github.io/svgomg/)
- Re-run the `npm run icons` scripts
